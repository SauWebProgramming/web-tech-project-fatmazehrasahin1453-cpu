# İnteraktif Medya Kitaplığı (SPA)

Bu proje, Sakarya Üniversitesi ISE-201 Web Teknolojileri dersi kapsamında geliştirilmiştir.

##  Proje Açıklaması
Uygulama; film, dizi ve kitap verilerini yerel bir JSON dosyasından fetch API kullanarak alan,
bu verileri modern ve duyarlı (responsive) bir arayüzde listeleyen bir Single Page Application (SPA)’dır.

Kullanıcılar:
- Medyaları kart görünümünde inceleyebilir
- Tür ve kategoriye göre filtreleme yapabilir
- Arama özelliğini kullanabilir
- Detay sayfasında seçilen medyanın bilgilerini görebilir
- Favorilere ekleyerek localStorage üzerinde saklayabilir

##  Kullanılan Teknolojiler
- HTML5 (Semantic HTML)
- CSS3 (Flexbox, Responsive Design)
- JavaScript (ES6+)
- Fetch API
- Local JSON
- LocalStorage
- GitHub Pages
